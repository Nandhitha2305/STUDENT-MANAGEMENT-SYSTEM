# STUDENT-MANAGEMENT-SYSTEM

** Overview**
The Student Management System is a simple console-based application developed in C++. It helps manage student records efficiently using Object-Oriented Programming (OOP) concepts. The project provides a menu-driven interface for performing basic operations on student data.

 **Features**
- Add a new student record
- Display all student records
- Search for a student by Roll Number
- Delete a student record
- Easy-to-use menu-driven interface

** Technologies Used
**
- C++
- Object-Oriented Programming (OOP)
- Standard Template Library (STL - Vector)

** Project Structure
**
- "main.cpp" – Contains the complete source code of the Student Management System.

** How to Run
**
1. Clone or download this repository.
2. Open the project in any C++ IDE (Code::Blocks, Visual Studio Code, Dev-C++, etc.).
3. Compile the "main.cpp" file.
4. Run the program and use the menu to manage student records.

** Concepts Used
**
- Classes and Objects
- Functions
- Vectors
- Loops
- Conditional Statements
- User Input and Output
